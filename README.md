Как запустить проект (Docker)

Мы упаковали весь проект в контейнеры, поэтому для запуска вам не нужно устанавливать Python или Flutter. Достаточно только Docker.

1) Установите и откройте Docker Desktop

2) Скачайте проект и откройте терминал в папке с файлом docker-compose.yml.

3) Запустите проект одной командой:
```
docker-compose up -d
```
Эта команда скачает необходимые образы (клиент и сервер) и запустит их в фоновом режиме.

4) Откройте приложение в браузере: Перейдите по адресу: http://localhost:60080
